Portable Spas Chat Widget Demo

This page demonstrates the embeddable chat widget. Scroll down for integration instructions.

How to Integrate

Copy and paste the following code into any webpage where you want the chat to appear:

<!-- Portable Spas Chat Widget -->
<div id="portable-spas-chat-widget" data-api-url="https://yourdomain.com/api/chat"></div>

<script src="https://yourdomain.com/chat-widget.js"></script>

Or use the inline version (no external files):

Copy the entire contents of chat-widget.html and paste it anywhere in your page.

Configuration:

  • data-api-url: Set this to your chat API endpoint (defaults to "/api/chat")
  • The widget is fully self-contained and won't interfere with existing styles
  • All CSS is scoped with aggressive overrides to prevent conflicts
